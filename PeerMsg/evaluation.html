PeerMsg – The Peer-to-peer Messaging System

PeerMsg is an efficient, easy to use, extensible message dissemination system for all small organisations that can't afford a good server.

It has a clean, fast GUI meant to give all users a pleasant communication experience.

Based on a peer-to-peer architecture, PeerMsg is faster and more reliable than traditional client-server message boards.
